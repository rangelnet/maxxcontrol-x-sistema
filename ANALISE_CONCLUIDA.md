## 🔍 Descobertas Principais

### MESH TV
- ✅ Usa tela de login (ActivityMac)
- ✅ Autentica com credenciais
- ✅ Retorna JWT token
- ✅ Armazena em SharedPreferences
- ✅ Usa em todas as requisições
- ✅ Suporta múltiplos usuários

### TV-MAXX-PRO
- ✅ Sem tela de login
- ✅ Token fixo no código
- ✅ Busca config na inicialização
- ✅ Mais rápido e simples
- ✅ Funciona bem para TV Box único

### Ambas
- ✅ Mesma arquitetura geral
- ✅ Painel Web (React)
- ✅ Backend API (Node.js)
- ✅ App Android (Compose/Kotlin)
- ✅ Banco de Dados (PostgreSQL)

- Não precisa de múltiplos usuários
- Token fixo é apropriado

- JWT token é mais seguro

- Escolha baseada em risco

# 📱 ADICIONAR MESH TV AO WORKSPACE

## 🎯 OBJETIVO
Adicionar o projeto Mesh TV ao workspace para eu poder analisar e comparar com o TV MAXX PRO.

## 📋 PASSO A PASSO

### No VS Code / Kiro:

1. **Abrir Command Palette:**
   - Pressione `Ctrl + Shift + P` (Windows)
   - Ou `Cmd + Shift + P` (Mac)

2. **Adicionar Pasta ao Workspace:**
   - Digite: `Workspaces: Add Folder to Workspace`
   - Pressione Enter

3. **Selecionar a Pasta do Mesh TV:**
   - Navegue até: `R:\Users\Usuario\Documents\tv-maxx\`
   - Selecione a pasta do Mesh TV
   - Clique em "Adicionar"

4. **Verificar:**
   - No Explorer (barra lateral esquerda)
   - Você deve ver 3 pastas agora:
     - MaxxControl
     - TV-MAXX-PRO-Android
     - [Nome da pasta do Mesh TV]

================================================================================
                              FLUXO DE LOGIN
================================================================================

1. USUÁRIO ABRE APP
   ↓
2. VERIFICA SE JWT TOKEN EXISTE
   ├─ Sim → Valida com backend
   │        ├─ Válido? → Home
   │        └─ Expirado? → Login
   └─ Não → Login
   ↓
3. TELA DE LOGIN (Layout mantido, cores TV-MAXX-PRO)
   ├─ Email: [usuario@email.com]
   ├─ Senha: [••••••••]
   └─ Clica "ENTRAR ▶"
   ↓
4. POST /api/auth/login
   ├─ Envia: email, password, device_id
   └─ Retorna: JWT token + user + config
   ↓
5. APP ARMAZENA TOKEN (SharedPreferences)
   ├─ token: "eyJhbGciOiJIUzI1NiIs..."
   ├─ user: "usuario@email.com"
   └─ expires_in: 3600
   ↓
6. APP BUSCA CONFIG E BRANDING
   ├─ GET /api/app-config/config
   ├─ GET /api/branding/active
   └─ GET /api/iptv-server/config
   ↓
7. APP NAVEGA PARA HOME
   ├─ Carrega canais
   └─ Renderiza interface
   ↓
8. USUÁRIO USA APP NORMALMENTE
   ├─ Seleciona canal
   ├─ Reproduz stream
   └─ Continua usando
   ↓
9. LOGOUT (DELETE /api/auth/logout)
   ├─ Remove token
   └─ Volta para login

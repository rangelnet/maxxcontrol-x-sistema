# 🚀 FAZER PUSH PARA GITHUB - AGORA

## ✅ DEPOIS DE INSTALAR O GIT

### 1. Fechar e Abrir PowerShell
- Feche o PowerShell se estiver aberto
- Pressione `Win + X`
- Clique em "Windows PowerShell"

### 2. Copie e Cole Estes Comandos (UM POR VEZ)

```powershell
cd "R:\Users\Usuario\Documents\tv-maxx\maxxcontrol-x-sistema"
```

```powershell
git status
```

```powershell
git add .
```

```powershell
git commit -m "Implementar sistema de branding completo com SQLite"
```

```powershell
git push origin main
```

## 🔑 SE PEDIR USUÁRIO/SENHA

**Usuário:** seu email do GitHub

**Senha:** NÃO é sua senha normal!
- Acesse: https://github.com/settings/tokens
- Clique "Generate new token (classic)"
- Marque: `repo`
- Clique "Generate token"
- Copie o token (começa com `ghp_...`)
- Cole como senha

## 🐛 PROBLEMAS?

**"git: command not found"**
→ Feche e abra o PowerShell novamente

**"Permission denied"**
→ Use token de acesso (veja acima)

**"not a git repository"**
→ Verifique se está na pasta correta
